# 🏦 Bank Management System

A simple web-based **Bank Management System** developed using **HTML, CSS, and JavaScript**.  
This project allows users to view and manage banking data such as customers, accounts, employees, branches, transactions, cards, and loans.

---

## 📌 Features

- Dashboard showing overall bank statistics
- View data for:
  - Branches
  - Departments
  - Employees
  - Customers
  - Accounts
  - Cards
  - Transactions
  - Loans
- Advanced analytics queries
- Interactive sidebar navigation
- Simple and user-friendly interface

---

## 🛠 Technologies Used

- **HTML** – Structure of the application
- **CSS** – Styling and layout
- **JavaScript** – Application logic
- **In-memory database (database.js)** – Stores sample data
